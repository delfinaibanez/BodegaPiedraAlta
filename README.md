# Piedra Alta – Landing Page de Bodega

Este proyecto consiste en una landing page para una bodega ficticia llamada **Piedra Alta**. Fue desarrollado como proyecto de portafolio para demostrar habilidades en desarrollo front-end utilizando HTML, CSS y JavaScript.

##  Características

- Diseño de landing page moderna
- Diseño responsive adaptable a distintos dispositivos
- Secciones estructuradas (inicio, historia, vinos, contacto)
- Interactividad básica con JavaScript

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

## 🎯 Objetivo del proyecto

El objetivo de este proyecto es mostrar habilidades en:

- Maquetación web
- Diseño responsive
- Estructuración de interfaces
- Desarrollo front-end sin frameworks
## ⚠️ Nota

La marca **Piedra Alta** es ficticia y fue creada únicamente con fines educativos y de portafolio.
